📡 API Endpoints

Rota Método Descrição Autenticação
`/` GET Página de login Não requerida
`/login` POST Autentica via LDAP Não requerida
`/dashboard` GET Lista de salas Obrigatória
`/api/salas` GET JSON com projetores Obrigatória
`/sala/{id}/status` GET Status de uma sala Obrigatória
`/sala/{id}/conectar` POST Conecta VNC Obrigatória
`/sala/{id}/desconectar` POST Desconecta VNC Obrigatória

🧩 Componentes do Sistema

Flask API

Servidor web leve que gerencia autenticação, sessões e controle do VNC. Roda na porta 80 do projetor.

LDAP Authenticator

Integração com o Active Directory da instituição. Valida credenciais em tempo real.

xtightvncviewer

Cliente VNC leve que exibe a tela do usuário no projetor. Controlado via subprocess.

UltraVNC Server

Servidor VNC que roda no notebook do usuário. Permite conexão reversa ao projetor.

Watchdog

Monitora saúde do sistema. Restarta serviços e limpa processos travados.

Guardian

Proteção contra falhas críticas. Verifica memória, CPU e reinicia se necessário.
