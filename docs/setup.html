🚀 Guia de Instalação

Configure o seu CaraProjetada passo a passo

Pré-requisitos

  • Obrigatório TV Box RK3229 com Armbian instalado
  • Obrigatório Acesso SSH ao dispositivo
  • Obrigatório Projetor/TV conectado via HDMI
  • Opcional Camera USB para streaming
  • Opcional Rede 172.17.0.0/16 configurada

Passo a Passo do Deploy

Passo 1: Conectar via SSH

ssh carapreta@172.17.28.179
# Senha: carapreta123

Passo 2: Instalar Dependências

sudo apt update
sudo apt install -y \
  python3 python3-flask python3-ldap3 \
  xtightvncviewer chromium-browser \
  x11-utils xserver-xorg-core xfwm4 lightdm \
  vlc vlc-bin

Passo 3: Configurar o Flask

Copie o arquivo e ajuste as configurações do AD:

# Copiar o app
cp caraprojetada/app/app.py /home/carapreta/

# Editar configuracoes do AD
nano /home/carapreta/app.py

# Ajustar:
AD_SERVER = 'ldap://10.198.1.2'
AD_DOMAIN = 'intranet.ufrb.edu.br'

Passo 4: Configurar Serviços

sudo cp caraprojetada/systemd/projetor.service /etc/systemd/system/
sudo cp caraprojetada/systemd/stream-cam.service /etc/systemd/system/

sudo systemctl daemon-reload
sudo systemctl enable --now projetor

# Se tiver camera:
sudo systemctl enable --now stream-cam

Passo 5: Scripts de Manutenção

# Copiar scripts
cp caraprojetada/scripts/*.sh /home/carapreta/
chmod +x /home/carapreta/*.sh

# Configurar cron
crontab -e
# Adicionar:
* * * * * /home/carapreta/totem_guardian.sh
* * * * * sleep 30 && /home/carapreta/totem_watchdog.sh
*/30 * * * * /home/carapreta/totem_watchdog.sh

Passo 6: Configurar Kiosk

# Testar manualmente:
export DISPLAY=:0
chromium --kiosk --start-maximized --noerrdialogs \
         --disable-infobars --incognito https://www.uol.com.br &

Passo 7: Configurar Senha VNC

echo "123456" > ~/.vnc_pass
chmod 600 ~/.vnc_pass

Verificação Pós-Deploy

# Verificar servicos
systemctl status projetor
systemctl status stream-cam

# Verificar porta 80
ss -tlnp | grep 80

# Verificar cron
crontab -l

# Testar autenticacao AD
python3 -c "
from ldap3 import Server, Connection, ALL
server = Server('ldap://10.198.1.2', get_info=ALL)
conn = Connection(server, user='teste@intranet.ufrb.edu.br', password='senha', authentication='SIMPLE')
print('AD OK' if conn.bind() else 'AD FALHOU')
"

Configuração Especial - Cose TV Box

A Cose TV Box requer configurações adicionais:

# Adicionar no /etc/hosts
echo "10.198.1.2 ldap.intranet.ufrb.edu.br" >> /etc/hosts

# Configurar DNS
echo "nameserver 10.198.1.2" > /etc/resolv.conf

# Reiniciar servicos
sudo systemctl restart projetor
sudo systemctl restart networking
