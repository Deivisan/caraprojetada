📚 Tutoriais

Aprenda a usar o CaraProjetada passo a passo

Tutorial 1: Fazer Login no Sistema

Guarde esse tutorial para referência rápida!

  • Acesse o Dashboard: Abra o navegador e acesse: https://dave-san.github.io/caraprojetada
  • Insira suas credenciais:
    • Usuário: Seu login institucional (ex: joao.silva)
    • Senha: Sua senha do domínio
  • Clique em "ACESSAR"
  • Pronto! Você será redirecionado para o dashboard com a lista de salas disponíveis.

Tutorial 2: Conectar seu Notebook ao Projetor

Passo 1: Preparar o Notebook

# Windows - Instale o UltraVNC
msiexec.exe /i UltraVNC-2.8.87.msi /VERYSILENT /SUPPRESSMSGBOXES

# Configure a senha (via registry ou manualmente)
# Senha: 123456

Passo 2: Iniciar o Servidor VNC

# Iniciar o serviço VNC
services.msc  # Procure por "UltraVNC Server"

# Ou via linha de comando
winvnc.exe -shared -accept_connections -desktop

Passo 3: Verificar IP do Notebook

ipconfig

# Anote o IPv4 (ex: 172.17.28.50)

Passo 4: Conectar via Dashboard

  1. Acesse o dashboard do CaraProjetada
  2. Selecione a sala desejada
  3. Clique em "CONECTAR TELA"
  4. Digite o IP do seu notebook quando solicitado
  5. Pronto! Sua tela aparecerá no projetor

Tutorial 3: Manutenção do Projetor

Verificar status dos serviços

ssh caraprojetada@172.17.28.179

# Verificar serviços
systemctl status projetor
systemctl status stream-cam

# Ver logs em tempo real
tail -f /var/log/syslog

Reiniciar serviços

# Reiniciar o serviço principal
sudo systemctl restart projetor

# Reiniciar o streaming (se tiver câmera)
sudo systemctl restart stream-cam

Reset de emergência

# Execute o script de reset
/home/carapreta/totem_reset.sh

# Ou manualmente:
sudo pkill -9 xtightvncviewer
sudo systemctl restart projetor

Tutorial 4: Configurando Cose TV Box

Esta seção é específica para a Cose TV Box, uma variação otimizada do CaraProjetada.

Passo 1: Identificar a TV Box na rede

# No dashboard, procure por "Cose TV Box" na lista de salas
# Ou use o IP atribuído pelo DHCP (ex: 172.17.28.200)

Passo 2: Configuração de rede

# Acesse a interface web da Cose TV Box
http://172.17.28.200:8080/config

# Configure:
- Nome da sala: "Sala 1 - Cose"
- IP do servidor AD: 10.198.1.2
- Domínio: intranet.ufrb.edu.br

Passo 3: Testar conexão

# Após configurar, reinicie a TV Box
sudo reboot

# Aguarde 2 minutos e acesse o dashboard
# A TV Box deve aparecer como "online"

Perguntas Frequentes

Como esquecer a senha do VNC?

A senha padrão é 123456. Para alterar, edite o arquivo ~/.vnc_pass no projetor.

O que acontece se o projetor não aparecer no dashboard?

Verifique: 1) Se o servidor está ligado, 2) Se está na mesma rede (172.17.x.x), 3) Se o serviço projetor está rodando.

Como transmitir apenas uma janela específica?

Use o recurso de seleção de área do UltraVNC. Clique com o botão direito e selecione "Select Area".

Posso usar outro cliente VNC?

Sim! O CaraProjetada funciona com qualquer cliente VNC compatível com o protocolo RFB. Recomendamos o TightVNC ou RealVNC.
