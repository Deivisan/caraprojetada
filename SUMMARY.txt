👮 SISTEMA DE WATCHDOG
═══════════════════════════════════════════════════════════════════════════

totem_guardian.sh (A cada 1 minuto)
  1. Verifica IP wlan0 (fallback dhclient)
  2. Verifica Xorg :0 (restart lightdm)
  3. Verifica xfwm4 (reinstala se necessário)
  4. Remove painéis XFCE residuais
  5. Força resolução 1920×1080
  6. Verifica Chromium kiosk
  7. Desliga screensaver

totem_watchdog.sh (A cada 30 minutos + fallback 1 min)
  1. Conectividade rede
  2. Status lightdm
  3. xfwm4 + xfce4-session
  4. Chromium rodando
  5. Resolução de vídeo

totem_reset.sh (Emergency)
  - Mata todos os processos gráficos
  - Reinicia lightdm
  - Verifica rede

⚠️ LIMITAÇÕES CRÍTICAS
═══════════════════════════════════════════════════════════════════════════

Kernel 4.4 (legacy)
  → Sem suporte a drivers novos
  → Bloqueador principal para melhorias

HTTP (sem HTTPS)
  → Tráfego não criptografado
  → Inadequado para produção segura

VNC Senha Fixa (123456)
  → Segurança inadequada
  → Sem configuração por sessão

Sem Rate Limiting
  → Brute force possível no login
  → Sem proteção contra DoS

RAM Limitada (962 MB)
  → Competição: Chromium + Flask + Xorg
  → Pode causar swapping em zram

eMMC Limitada (7.3 GB)
  → Pouco espaço para armazenamento
  → Necessário monitoramento

Resolução: Mismatch
  → Guardian força 1920×1080
  → Display real é 1360×768
